# 🐶 Dogs

Este é um projeto desenvolvido durante o curso de **React Completo** da **Origamid**. O objetivo principal foi construir uma aplicação web completa utilizando **React**, explorando desde os fundamentos da biblioteca até conceitos como autenticação, consumo de APIs, gerenciamento de estado e criação de componentes reutilizáveis.

> **Nota:** Este é um sistema fictício criado exclusivamente para fins didáticos e para aprimorar minhas habilidades no desenvolvimento de aplicações front-end com React.

---

## 🚀 Tecnologias Utilizadas

O projeto foi desenvolvido com foco nas principais ferramentas do ecossistema React:

- **React**
- **React Router DOM**
- **JavaScript (ES6+)**
- **HTML5**
- **CSS3**
- **Vite**
- **REST API**

---

## ✨ Funcionalidades

- 🔐 Autenticação de usuários
- 👤 Cadastro de novos usuários
- 🔑 Recuperação e alteração de senha
- 🖼️ Publicação de fotos
- ❤️ Visualização do feed de publicações
- 👀 Exibição de detalhes e estatísticas das fotos
- 📄 Perfil do usuário
- 📱 Interface responsiva
- ⚡ Carregamento otimizado com Skeleton Loading
- 🏷️ Atualização dinâmica de título e descrição das páginas (SEO)

---

## 📚 Conceitos Praticados

Este projeto foi desenvolvido com o objetivo de consolidar conhecimentos essenciais sobre React, incluindo:

- Componentização
- Hooks (`useState`, `useEffect`, `useContext`, `useCallback`, `useRef`)
- Criação de Hooks Customizados
- Context API
- Rotas com React Router
- Consumo de APIs REST
- Requisições assíncronas com `fetch`
- Formulários controlados
- Validação de campos
- Gerenciamento de autenticação
- Manipulação do DOM com React
- Organização escalável de projetos
- Boas práticas de desenvolvimento

---

## 🎯 Objetivos do Projeto

Além de reproduzir uma aplicação completa proposta durante o curso, este projeto teve como objetivo aprofundar o entendimento sobre o ecossistema React, reforçando conceitos fundamentais para o desenvolvimento de interfaces modernas, reutilizáveis e escaláveis.

---

## 🎨 Design e Identidade Visual

O projeto apresenta uma interface limpa e minimalista, priorizando a experiência do usuário por meio de componentes reutilizáveis, animações sutis, feedback visual durante carregamentos e uma navegação intuitiva.
